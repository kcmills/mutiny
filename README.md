# Mutiny [![Build Status](https://travis-ci.org/louismrose/mutiny.png?branch=master)](https://travis-ci.org/louismrose/mutiny)

A tiny mutation testing framework.

#### Usage
* `git clone` this repo
* `bundle install`
* `ruby lib/driver.rb examples/<EXAMPLE_NAME>`
